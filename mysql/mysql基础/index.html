表的操作

主要类型: INT(整数),CHAR/VARCHAR(字符串),DATE(日期)

CHAR和VARCHAR的区别:

CHAR长度固定，设了CHAR(10)的情况，不满会用空格补足到10位
VARCHAR长度可变，如数据长度为X，实际存储为X+1/2位(标识你输入的长度,超过255用2位,小于则1位)

主要参数: NOT NULL(不能为空),AUTO_INCREMENT(自增),PRIMARY KEY(主键)
